# 2.1 数据与文字的表示方法
## 数据
表示方法总体上两种：**定点/浮点**
### 定点表示
> 通常表示纯整数 / 纯小数
$[x_n][x_{n-1}...x_0]$
其中$x_n$表示符号

### 浮点表示
> $[E_s][E_{m-1}...E_0][M_s][M_{n-1}...M_0]$
阶符阶码数符尾数

### 机器码
> 现在区分真值与机器码 真值就是数的值，可以写做二进制也可以是其他进制，带符号，如+8 +100 ...
机器码就是机器码

#### 原码表示
$正数: [x]_原 = x$
$负数: [x]_原 = 2^n - x$

> 这里的x指代真值，今后也是

like: $2^n = 100000..0$
$ 2^n - x = 1111, let :x = -7, n = 3$
